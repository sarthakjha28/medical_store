# Mini E-commerce Platform

## Project Overview
This is a mini e-commerce web application developed using HTML, JavaScript, FastAPI, and MySQL.
It allows users to register, login, view products, and add items to cart.
An admin panel is also included to manage products.

## Tech Stack
- HTML
- CSS
- JavaScript
- Python (FastAPI)
- MySQL
- Fetch API

## Features
### User Side
- User Registration
- User Login
- View Products
- Add to Cart
- Cart Count Update

### Admin Side
- Admin Login
- Add Products
- Delete Products

## Project Structure
- backend/
- fronend/
- images/

## How to Run
1. Run FastAPI backend
2. Open frontend HTML files in browser

## API Endpoints
- POST /register
- POST /login
- GET /products
- POST /add-to-cart
